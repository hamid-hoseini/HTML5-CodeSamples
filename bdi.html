The HTML tag is used on a span of text that is to be isolated from its surroundings for the purposes of bidirectional text formatting. This can be useful when displaying right-to-left text (such as Arabic) inside left-to-right text (such as English) when the text-direction is unknown. The element allows you to honor the correct directionality of text when this is unknown (such as in the case with user-generated content).
